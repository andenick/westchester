## Data Source 1: Tax Parcel GIS Data

### Westchester County GIS - Tax Parcels

**Primary Source**: Westchester County GIS Portal
**URL**: https://gis.westchestergov.com/datasets/wcgis::tax-parcels/explore

**What it provides**:
- 350,000+ individual tax parcels
- Parcel IDs, addresses, lot sizes
- Assessed values (land + improvement)
- Municipality/jurisdiction
- Property classes
- Geographic boundaries (polygons)

**Download Options**:

1. **Manual Download** (Recommended for initial attempt):
   - Visit: https://gis.westchestergov.com/datasets/wcgis::tax-parcels/explore
   - Click "Download" button
   - Select format: GeoJSON, Shapefile, or CSV
   - Download full dataset

2. **ArcGIS REST API** (Programmatic):
   - Endpoint: https://gis.westchestergov.com/arcgis/rest/services/
   - Service: Tax Parcels Feature Layer
   - Can query programmatically with pagination
   - Requires ArcGIS API knowledge

3. **Alternative: NYS GIS Clearinghouse**:
   - URL: https://gis.ny.gov/
   - Search for "Westchester tax parcels"
   - May have older/archived versions

**Expected File Size**: ~500 MB (GeoJSON) or ~200 MB (Shapefile)

**Data Quality**: Official county GIS data, updated regularly

## Data Source 2: Municipal Tax Profiles

### NYS Office of Real Property Tax Services

**Portal URL**: https://www.tax.ny.gov/research/property/reports.htm

**Alternative Portal**: https://pad.tax.ny.gov/index

**What it provides**:
- Municipal tax rates by year
- Levy limits and compliance
- Assessment ratios (market value vs assessed value)
- Equalization rates
- Tax roll summaries

**Manual Process Required**:

## Data Source 3: U.S. Census Bureau

### American Community Survey (ACS)

**URL**: https://data.census.gov/

**What it provides**:
- Median home values by census tract
- Home ownership rates
- Property value distributions
- Demographic correlations with property values

**Access Methods**:
1. **Web Interface**: data.census.gov (manual selection)
2. **API**: https://api.census.gov/data.html (programmatic)
3. **Bulk Downloads**: https://www2.census.gov/programs-surveys/acs/

**Tables Needed**:
- B25077: Median Home Value
- B25075: Value of Owner-Occupied Units
- B25024: Units in Structure
- B25003: Tenure (Owner/Renter Occupied)

**Census Tracts in Westchester**: 241 (already have boundaries)

## Implementation Plan

### Option A: Quick Start (Parcel Data Only)

**Time**: ~30 minutes

1. Download tax parcel GeoJSON from Westchester GIS
2. Create Python script to aggregate by municipality:
   - Count parcels per municipality
   - Calculate average/median assessments
   - Extract property class distributions
3. Generate JSON summary file
4. Update PropertyTaxDashboard.tsx to load JSON

**Time**: 4-6 hours

1. Download tax parcel GeoJSON (30 min)
2. Download 50 municipal tax profile PDFs (2-3 hours)
3. Extract data from PDFs using pdfplumber (1-2 hours)
4. Fetch Census ACS data via API (30 min)
5. Integrate all sources into comprehensive dataset (1 hour)
6. Update dashboard with full historical trends

**Deliverable**: Complete real data (counts, assessments, trends, rates)
