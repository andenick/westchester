## 🔧 Troubleshooting

### Common Issues

**Issue: Map not loading**
- **Solution:** Check that backend is running (http://localhost:8000/api/health)
- **Solution:** Verify data files exist in `Technical/data/raw/infrastructure/`

**Issue: No data on charts**
- **Solution:** Run data download: `python scripts/download_all_data.py`
- **Solution:** Check API endpoint: http://localhost:8000/api/demographics/county

**Issue: 404 errors in browser console**
- **Solution:** Ensure backend API is running on port 8000
- **Solution:** Check CORS settings in FastAPI main.py

**Issue: Slow map rendering**
- **Solution:** Toggle off some layers to reduce load
- **Solution:** Zoom in to reduce visible features
